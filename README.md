🔁 Two Pointer Technique – Interview Revision Guide

## What is the Two Pointer Technique?
The Two Pointer Technique is a common algorithmic approach used in Data Structures and Algorithms to solve problems efficiently by using two indices (pointers) to traverse a data structure, usually an array or string.
Instead of using nested loops (O(n²)), the two pointer approach helps reduce time complexity to O(n) in many problems, making it highly useful for coding interviews and competitive programming.
This technique is mainly used in:
- Sorted arrays
- Strings
- Sliding window problems
- Searching pairs or subarrays
- Partitioning problems
  ## Why Learn Two Pointers?
- Reduces time complexity
- Frequently asked in coding interviews
- Improves problem-solving efficiency
- Core pattern in DSA rounds (placements & hackathons)
  
## Main Types of Two Pointer Patterns
The Two Pointer Technique can be broadly classified into 3 major types:

## 🔹 Type 1: Opposite Direction Pointers (Start & End)
In this pattern, one pointer starts from the beginning and the other from the end of the array.

📍 When to Use:
- Sorted arrays
- Pair sum problems
- Palindrome checking
- Removing duplicates

🧩 Example Problems:
- Two Sum (Sorted Array)
- Container With Most Water
- Valid Palindrome
- Remove Duplicates from Sorted Array

## 🔹 Type 2: Same Direction Pointers (Fast & Slow Pointers)
Both pointers move in the same direction but at different speeds or positions.

📍 When to Use:
- Removing elements in-place
- Cycle detection
- Sliding window optimizations
- Partitioning arrays

🧩 Example Problems:
- Remove Element
- Move Zeroes
- Linked List Cycle Detection
- Longest Substring Without Repeating Characters

## 🔹 Type 3: Sliding Window (Dynamic Two Pointers)
This is an advanced form of two pointers where a window expands and shrinks based on conditions.

📍 When to Use:
- Subarray problems
- Longest/Shortest window problems
- String pattern matching
- Optimization problems

🧩 Example Problems:
- Longest Substring Without Repeating Characters
- Minimum Window Substring
- Maximum Sum Subarray of Size K
- Fruits into Baskets
